#This is an API project, based on the video by the lovely Ania Kubow (YT link: https://www.youtube.com/watch?v=GK4Pl-GmPHk)

##Used Dependencies
Cheerio for scraping data, Axios, Nodemon and Express.

##How to run it
Clone the repository, run npm i to install all dependancies and in the console type: npm run start and it will start the server.

Hopefully this will turn into a bigger project!
